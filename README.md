# Portafolio de Victor Muñoz

## Descripción

Portafolio profesional de Victor Muñoz, estudiante de 21 años de la carrera ciberseguridad y egresado electricidad de Concepción (Chile).
El sitio web muestra sus habilidades técnicas, proyectos enfocados en seguridad, certificaciones y experiencia profesional.

El portafolio incluye secciones como:
- Sobre mí: Biografía y datos personales
- Habilidades: Competencias técnicas, informáticas y eléctricas
- Proyectos: 6 proyectos de seguridad incluyendo herramientas de análisis de malware, encriptación y gestión de inventario
- Certificados: Certificación IT y eléctricas
- Contacto: Formulario de contacto e información

## Tecnologías Utilizadas

- **HTML5**: Estructura semántica del sitio
- **CSS3**: Estilos modernos con gradientes, flexbox, grid, animaciones y diseño responsivo
- **Google Fonts**: Tipografías personalizadas (Sekuya, Playwrite IE, Open Sans, Miranda Sans)
- **Diseño Responsivo**: Optimizado para móviles con media queries

## Cómo Ejecutar

1. Descarga o clona el repositorio
2. Abre el archivo `index.html` en cualquier navegador web moderno
3. El sitio es completamente autocontenido y no requiere servidor
